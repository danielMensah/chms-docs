---
title: "Groups"
description: "Create groups, manage rosters, assign leaders, bulk assign members, and link groups to ministries in Bethel ChMS."
---

Groups organise members into fellowship structures such as small groups, zones, classes, or cells. Your branch can rename the group type, so the app sidebar may show a branch-specific label such as **Cells** or **Zones** instead of **Groups**.

<Info>
**Who can do this:** Org Owner, Org Admin, Branch Admin, and Editor can create and edit groups and rosters. Org Owner, Org Admin, and Branch Admin can delete groups and reassign all members.
</Info>

Members can belong to more than one active group. Use each group's roster to confirm membership. The member profile's **Group** tab currently shows a single group assignment and should not be treated as the full list of all active group memberships.

## Create a group

<Steps>
  <Step title="Open Groups">
    In the sidebar, click **Groups** or your branch's custom group label.
  </Step>
  <Step title="Start a group">
    Click **Add Group**. If your branch uses a custom label, the button uses that label.
  </Step>
  <Step title="Enter details">
    Enter **Name**. Optionally enter **Description** and choose a **Ministry**.
  </Step>
  <Step title="Create the group">
    Click **Create Group**.
  </Step>
</Steps>

Group names must be unique within the branch. New groups start as **Active**.

## Link a group to a ministry

You can link a group to an active ministry when you create or edit the group. When you link a group, Bethel ChMS can add the group's active members to the ministry roster through the group relationship.

<Note>
Only active, non-archived ministries in the same branch are available in the **Ministry** selector.
</Note>

The group detail page shows a **Ministry** badge. Click the ministry name to open the linked ministry.

## Find and filter groups

The group list shows each group's name, status, member count, leader, and created date.

Use the status filter to show:

- **All**
- **Active**
- **Inactive**

Use **Sort by** or the table column headers to sort by name, member count, or created date.

## Add a member

<Steps>
  <Step title="Open the group">
    Click the group name in the group list.
  </Step>
  <Step title="Click Add Member">
    Click **Add Member** in the **Roster** section.
  </Step>
  <Step title="Choose a member">
    Search for a member in the same branch and select them.
  </Step>
  <Step title="Add the member">
    Click **Add Member**.
  </Step>
</Steps>

Members can belong to more than one group. The selector shows a member's current groups to help you avoid accidental duplicates.

## Bulk assign members

Use **Bulk Assign** when you need to add many members to the same active group.

1. Open an active group.
2. Click **Bulk Assign**.
3. Search or select from members who are not already in this group.
4. Click **Assign Members**.

Bulk assignment adds selected members with the **Member** role. Inactive members are not included in the available list.

## Change roles

Each group roster member has one role:

- **Leader**
- **Assistant Leader**
- **Member**

To change a role, open the group, find the person in **Roster**, and click **Change Role**.

## Remove a member

Removing a member closes their active group membership. It does not delete the member record or attendance history.

1. Open the group.
2. Find the person in **Roster**.
3. Click **Remove**.
4. Confirm the removal.

## Reassign all members

Branch Admin and org-level admin roles can move all active members from one group to another active group in the same branch.

1. Open the source group.
2. Click **Reassign All Members**.
3. Select the target group.
4. Click **Move Members**.

All moved members receive the **Member** role in the target group.

## Edit, deactivate, or delete a group

Use the group list **Actions** menu to edit a group's name, description, linked ministry, or active status.

Inactive groups remain visible when you filter for **Inactive**, but you cannot add members or bulk assign members to an inactive group.

<Warning>
Deleting a group soft-deletes the group and closes active memberships. Use **Reassign All Members** first if you need to move the roster before deleting.
</Warning>

## Close events

The group detail page shows **Close Events** when events are linked to the group. It separates upcoming events from recent past events.

Use event setup to link an event to a group.
